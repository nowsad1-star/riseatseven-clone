# Rise at Seven Homepage Clone

Frontend recreation task for Junior MERN Developer position at DeveloperLook.

## Project Overview
This project recreates the homepage of Rise at Seven as closely as possible, including:

- Homepage layout
- Mobile responsive version
- Hamburger menu behavior
- Responsive sections and footer

## Built With
- HTML5
- CSS3
- JavaScript

## Features
- Responsive navigation
- Mobile hamburger menu
- Hero section
- Services section
- Work/Case Studies section
- Footer CTA
